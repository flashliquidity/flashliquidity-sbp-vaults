# FlashLiquidity SBP-Vaults

Liquidity vaults for self-balancing pools.

## Requirements

- [foundry](https://book.getfoundry.sh/getting-started/installation)

## Install Dependencies

`yarn setup`

## Compile Contracts

`yarn compile`

## Run Full Test Suite

`yarn test`

## Run Unit Tests

`yarn unit-test`

## Run Fuzz Tests

`yarn fuzz-test`

## Run Invariant Tests

`yarn invariant-test`

## Run Coverage

`yarn coverage`

## Gas Report

`yarn gas-report`

## Run Slither

`yarn slither`
